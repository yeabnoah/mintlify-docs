# Troubleshooting

- **Missing environment variables:**  
  If you need backend variables, refer to the main project guide.

- **Component import errors:**  
  Double-check your import paths and file names.

- **Landing page not displaying:**  
  Ensure you have configured `page.tsx` as shown in [Landing Page Variants](/frontend/landing-pages).

- **CMS errors:**  
  Verify your CMS base URL is set and accessible.
