# Customization Tips

- Mix and match any components from `@/features/home/` for your own landing page.
- Use Tailwind utility classes for quick style adjustments.
- All components are dark-mode only by default.
- For advanced theming, edit `theme-provider.tsx` and your Tailwind configuration.
