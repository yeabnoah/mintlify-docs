# Component Library & Usage

> **Component Structure:**
> - Most landing and feature components are in `@/features` (e.g., `@/features/home/`, `@/features/content/`).
> - General-purpose, UI, and utility components are in `ui/` (e.g., `ui/button`, `ui/card`).

## General & UI Components (`ui/`)

- **Navigation:** `nav-main`, `nav-user`, `nav-projects`, `app-sidebar`
- **Dashboard:** `usage-dashboard`, `usage-meter`, `team-switcher`, `user-menu`
- **Forms:** `login-form`, `prompt-input`
- **UI Primitives:** All components in `ui/` (e.g., `button`, `input`, `card`, `dialog`, `dropdown-menu`, `tabs`, `table`, `badge`, `avatar`, etc.)
- **Skeletons/Loaders:** `page-loading`, `skeleton`, `loader`
- **Content:** `faq`, `cta`, `footer`, `header`, `blog`, `privacy-policy`
- **AI:** `ai-conversation`, `ai-animated-chat`, `ai-chat`

**Usage Example:**

```tsx
import { Button } from "ui/button";
<Button variant="outline">Click me</Button>
```
