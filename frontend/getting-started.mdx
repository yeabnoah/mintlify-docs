# Getting Started

1. **Install Dependencies**

   ```bash
   bun install
   ```

2. **Configure Environment Variables**

   See [Environment Variables](/frontend/environment).

3. **Run the Development Server**

   ```bash
   bun dev:web
   ```

The app will be available at [http://localhost:3001](http://localhost:3001).
