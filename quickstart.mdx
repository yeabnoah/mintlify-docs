## âš¡ Quick Setup

### 1. Clone the Repository

<CodeGroup>
```bash npm
git clone https://github.com/your-org/ultimate-cursor-ai-boilerplate.git
cd ultimate-cursor-ai-boilerplate
```
</CodeGroup>

### 2. Install Dependencies

<CodeGroup>
```bash npm
npm install
```

```bash bun
bun install
```
</CodeGroup>

### 3. Environment Setup

Create two `.env` files in your project:
#### Frontend [apps/web] (.env)

Copy the example environment file and update with your values:

```bash
cd apps/web && cp .env.example .env
```


<CodeGroup>
```env .env
# Core
NEXT_PUBLIC_APP_URL=http://localhost:3001
NEXT_PUBLIC_API_URL=http://localhost:3000
```
</CodeGroup>

#### Backend [apps/server] (.env)

Copy the example environment file and update with your values:

```bash
cd apps/server && cp .env.example .env
```


<CodeGroup>
```env .env
# Core
CORS_ORIGIN=your_frontend_url

# Database

DATABASE_URL=your_supabase_database_url (transa)
DIRECT_URL=your_supabase_direct_url

# Auth
BETTER_AUTH_SECRET=your_better_auth_secret
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret

# Services
STRIPE_SECRET_KEY=your_stripe_secret_key
GOOGLE_AI_API_KEY=your_google_ai_api_key
STRIPE_PRODUCT1_PRICE_ID=your_stripe_product_id

# email
RESEND_API_KEY=your_resend_api_key

# OPEN ROUTER AI
OPENROUTER_API_KEY=your_openrouter_api_key

```
</CodeGroup>

### 4. Database Setup

```bash
 cd apps/server
```


<CodeGroup>
```bash bun
# Generate Prisma client
npx prisma generate

# Run migrations
npx prisma db push
```
</CodeGroup>

### 5. Start Development Server

<CodeGroup>
```bash bun
bun dev
```
</CodeGroup>

Visit `http://localhost:3001` to see your frontend application. <br />
Visit `http://localhost:3000` to see your backend application. <br />
Visit `http://localhost:3002` to see your email application. <br />
Visit `http://localhost:3003` to see your cms application. <br />
