## âš¡ Quick Setup

### 1. Clone & Install

<CodeGroup>
```bash bash
git clone https://github.com/your-org/ultimate-cursor-ai-boilerplate.git
cd ultimate-cursor-ai-boilerplate
bun install
```
</CodeGroup>

### 2. Environment Setup

Create `.env` files in both `apps/web` and `apps/server` directories:

#### Frontend [apps/web/.env]
```env
NEXT_PUBLIC_SERVER_URL=http://localhost:3000
NEXT_PUBLIC_APP_URL=http://localhost:3001

NEXT_PUBLIC_POLAR_PRODUCT_ID=example_value // If you choose polar as payment provider
```

#### Backend [apps/server/.env]
```env
CORS_ORIGIN=http://localhost:3001

BETTER_AUTH_SECRET=
BETTER_AUTH_URL=http://localhost:3000
GOOGLE_GENERATIVE_AI_API_KEY=


DATABASE_URL=
DIRECT_URL=


POLAR_ACCESS_TOKEN= // If you choose polar as payment provider
POLAR_SUCCESS_URL= // If you choose polar as payment provider

RESEND_API_KEY= // If you choose resend as email provider

STRIPE_SECRET_KEY=

STRIPE_WEBHOOK_SECRET=

STRIPE_EXAMPLE_PRODUCT=

OPENROUTER_API_KEY=
```

For detailed configuration of each feature, see the corresponding sections in Next Steps.

### 3. Database Setup

```bash
cd apps/server
npx prisma generate
npx prisma db push
```

### 4. Start Development

<CodeGroup>
```bash bun
bun dev
```
</CodeGroup>

Visit `http://localhost:3001` for the frontend application.
