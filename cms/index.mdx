## âš¡ï¸ Setting Up the CMS

1. **Install Dependencies**

   ```bash
   cd apps/cms
   bun install # or: pnpm install
   cp .env.example .env
   ```

2. **Set Up Your Database**

   :::tip
   The CMS requires a PostgreSQL database. We recommend [Supabase](https://app.supabase.com/) (free), but you can use any Postgres instance.
   :::

   - [Database setup instructions â†’](/database/cms-setup)

3. **Configure Database Connection**

   Open `.env` and set the `DATABASE_URL` to your Supabase or Postgres connection string.

4. **Start the CMS**

   ```bash
   bun turbo run dev --filter cms
   # or: pnpm turbo run dev --filter cms
   ```

5. **Access the CMS**

   Open [http://localhost:3003](http://localhost:3003) in your browser.

   Your CMS will be live on port 3003! ğŸ‰

## âš™ï¸ CMS Configuration

- Open `.env` in your code editor.
- Set your `DATABASE_URL` to your database connection string.
- For enhanced security, set `PAYLOAD_SECRET` to a random string. You can run the following command and paste the result in `PAYLOAD_SECRET`:

  ```bash
  openssl rand -hex 32
  ```
